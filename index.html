Installatie (kort)

  1. Upload deze bestanden naar een GitHub repo (index.html + game-bestanden).
  2. Activeer GitHub Pages (Settings → Pages → kies branch 'main' en root '/').
  3. Je site is bereikbaar via https://<gebruikersnaam>.github.io/<repo>/

Veiligheid & privacy

Gebruik alleen client-side opslag (localStorage) voor saves en verzamel geen persoonlijke data. GitHub Pages serveert via HTTPS automatisch.
Content Security Policy is ingesteld via <meta> tag om externe scripts te blokkeren.

Snelle GitHub Pages handleiding

De snelste manier (web):
1. Maak een GitHub account (github.com). 
2. Maak een nieuwe repository (naam: mijn-games).
3. Klik op "Add file" → "Upload files" en sleep je index.html en game .html(s) in.
4. Commit changes.
5. Ga naar Settings → Pages → Source: branch 'main' (root) → Save.
6. Wacht enkele minuten; site is live op https://<gebruikersnaam>.github.io/<repo>/
Of gebruik Git (voor wie Git kent):
git init
git add .
git commit -m "initial"
git branch -M main
git remote add origin https://github.com/gebruikersnaam/reponame.git
git push -u origin main
